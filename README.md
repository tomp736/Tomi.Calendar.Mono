# Tomi.Calendar.Mono
Playground for learning blazor by writing a calendar/scheduler/notetaking app.

Using 
- Blazor Wasm
- Fluxor for Centralized State
- Blazored Components (Modal, LocalStorage, TextEditor)
- SignalR for server push Desktop Notification
